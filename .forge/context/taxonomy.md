# Taxonomy

*Navigation: [Project Overview](project-overview.md) | [Architecture](architecture.md) | [Conventions](conventions.md)*

## Elements
The core building blocks of AI architecture, categorized into groups (columns) and maturity levels (rows).

### Groups (Columns)
- **Primitives (Pr)**: Foundational capabilities (e.g., Prompting)
- **Models (Md)**: Core reasoning engines (e.g., LLMs, Vision Models)
- **Reactive (Re)**: Single-turn execution elements
- **Retrieval (Rt)**: Elements that fetch and manage context (e.g., Vector DBs, Embeddings)
- **Orchestration (Or)**: Elements that coordinate tasks (e.g., Agents, Frameworks)
- **Validation (Va)**: Elements ensuring safety and correctness (e.g., Guardrails)

### Maturity Levels (Rows)
- **Foundation**: Well-established, standard elements
- **Mainstream**: Widely adopted in production
- **Emerging**: Cutting-edge, actively developing capabilities

## Recipes
Pre-defined architectural combinations of elements that solve specific use cases (e.g., "RAG Knowledge Base", "Multi-Agent Swarm").
