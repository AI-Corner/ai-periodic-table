# AI Periodic Table — Project Overview

*Navigation: [Architecture](architecture.md) | [Conventions](conventions.md) | [Taxonomy](taxonomy.md)*

## What It Does
An interactive web application that organizes AI components (Primitives, Models, Agents, etc.) into a "Periodic Table" format. It serves as a visual learning and design tool, offering architecture recipes, enterprise audit checklists, and simple explanations to help users understand AI system architectures from basic chatbots to complex multi-agent swarms.

## Tech Stack
- **Frontend Framework**: React 18
- **Build Tool**: Vite
- **Styling**: Vanilla CSS
- **Deployment**: GitHub Pages

## Project Scope
- **In Scope**: Interactive UI, component explanations, architecture recipes, ELI5 descriptions, enterprise audit lists.
- **Out of Scope**: Backend services, database integrations, user authentication, live execution of AI components.
